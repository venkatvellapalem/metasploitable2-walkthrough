

# Metasploitable2 Documentation

A collaborative repository documenting reconnaissance, enumeration, exploitation, privilege escalation, and reporting against the intentionally vulnerable Metasploitable 2 virtual machine.

## Objectives

- Learn penetration testing methodology
- Document every vulnerable service
- Understand exploitation techniques
- Produce professional reports
